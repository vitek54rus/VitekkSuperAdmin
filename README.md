## 🚀 О проекте

**VITEKK SUPER ADMIN** - это профессиональная веб-ориентированная система управления базами данных PostgreSQL, которая позволяет:

- Управлять множеством баз данных через единый интерфейс
- Создавать и удалять таблицы с выбором типов данных
- Выполнять SQL запросы с сохранением истории
- Экспортировать данные в JSON и CSV форматы
- Работать с данными через интуитивно понятный UI

---

## ✨ Возможности

### 🗄️ Управление базами данных
- Подключение к нескольким серверам PostgreSQL
- Создание новых баз данных прямо из интерфейса
- Переключение между базами данных
- Сохранение подключений в файле `connections.json`

### 📊 Управление таблицами
- Создание таблиц с выбором типов данных
- Просмотр всех таблиц в базе данных
- Удаление таблиц с подтверждением
- Автоматическое определение структуры таблиц

### 📝 Работа с данными
- Добавление новых записей через форму
- Редактирование существующих записей (двойной клик)
- Удаление записей (одной или массово)
- Копирование/дублирование записей
- Массовое удаление с выбором чекбоксами

### 💻 SQL Редактор
- Выполнение любых SQL запросов
- История последних 20 запросов
- Сохранение часто используемых запросов
- Экспорт результатов в JSON/CSV
- Быстрые примеры запросов

### 🎨 Интерфейс
- Темный спортивный дизайн
- Адаптивная верстка
- Удобные модальные окна
- Подсветка синтаксиса SQL

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/vitek54rus/VitekkSuperAdmin.git
cd VitekkSuperAdmin
```

### 2. Настройка бэкенда

```bash
# Перейдите в папку бэкенда
cd backend

# Установите зависимости
pip install fastapi uvicorn sqlalchemy psycopg2-binary pydantic
```

### 3. Запуск бэкенда

```bash
python main.py
```

**Ожидаемый вывод:**
INFO: Uvicorn running on http://127.0.0.1:8000
INFO: Application startup complete.


### 4. Настройка фронтенда

*Откройте НОВЫЙ терминал!*

```bash
# Перейдите в папку фронтенда (из корня проекта)
cd frontend

# Установите зависимости
npm install

# Запустите приложение
npm start
```

### 5. Открытие приложения

Перейдите в браузере: [http://localhost:3000](http://localhost:3000/)

### 6. Подключение к базе данных

1. Нажмите **CONNECT**
2. Нажмите **+ ADD DATABASE**
3. Заполните поля:
   - **Connection Name**: `MyDB` (любое имя)
   - **Host**: `localhost`
   - **Port**: `5432`
   - **Database Name**: `postgres`
   - **Username**: `postgres`
   - **Password**: ваш пароль
4. Нажмите **ADD**
5. Выберите созданную БД

## 📖 Использование

### Управление базами данных

| Действие | Как сделать |
|----------|-------------|
| Добавить БД | Нажать **+ ADD DATABASE** → заполнить форму |
| Создать новую БД | Нажать **+ CREATE NEW DB** → ввести имя |
| Выбрать БД | Кликнуть по имени БД в списке |
| Удалить БД | Нажать **DELETE** рядом с БД |

### Управление таблицами

| Действие | Как сделать |
|----------|-------------|
| Создать таблицу | Нажать **CREATE** → заполнить форму |
| Выбрать таблицу | Кликнуть по имени таблицы |
| Удалить таблицу | ПКМ по таблице → подтвердить |
| Обновить список | Нажать **REFRESH** |

### Работа с данными

| Действие | Как сделать |
|----------|-------------|
| Добавить запись | Нажать **ADD RECORD** → заполнить форму |
| Редактировать | Двойной клик по строке или кнопка **EDIT** |
| Копировать | Нажать **COPY** |
| Удалить | Нажать **DEL** или ПКМ → Delete |
| Массовое удаление | Выбрать чекбоксы → нажать **DELETE N** |

## 💻 SQL Редактор

| Функция | Описание |
|---------|----------|
| **EXECUTE QUERY** | Выполнить SQL запрос |
| **SAVE** | Сохранить текущий запрос |
| **SAVED** | Показать сохраненные запросы |
| **HISTORY** | Показать историю запросов |
| **EXPORT** | Экспортировать результат в JSON/CSV |
| **CLEAR** | Очистить редактор |
